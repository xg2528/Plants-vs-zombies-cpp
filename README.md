# 植物大战僵尸 - C++ 简易还原版  
## Plants vs. Zombies - Simple C++ Recreation

一个使用 C++ 和 EasyX 图形库开发的《植物大战僵尸》基础版本，旨在练习游戏开发与面向对象编程和vibe coding。  
A basic clone of *Plants vs. Zombies* developed with C++ and EasyX graphics library, for practicing game development and OOP,as well as vibe coding.

---

### 游戏特性 / Features
- 主菜单与滚动背景动画  
- 植物卡槽：豌豆射手、向日葵  
- 阳光收集与僵尸生成  
- 基础碰撞检测与攻击逻辑  
- 关卡波次管理系统  
- 胜利 / 失败判定  

- Main menu with scrolling background animation  
- Plant selection: Peashooter, Sunflower  
- Sun collection and zombie spawning  
- Basic collision detection and attack logic  
- Wave management system  
- Win / loss conditions  

---

### 如何运行 / How to Run

1. **环境要求**  
   - Visual Studio 2022（或更高版本）  
   - EasyX 图形库（[https://easyx.cn](https://easyx.cn)）  

2. **编译与运行**  
   - 打开 `pvz/pvz.sln` 解决方案文件  
   - 设置 `pvz` 为启动项目  
   - 编译并运行（Debug / Release 均可） (release 模式有问题) 

> 注意：首次运行前请确保将资源文件夹 `resource/` 放置在可执行文件所在目录。
